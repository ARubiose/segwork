Welcome to Segwork’s documentation!

Segwork is a Python library that includes a set of tools for semantic segmentation.

Note

This project is part of a Master Thesis for the European Master in Software Engineering (EMSE).

Installation

To use Segwork, first install it using pip:

(.venv) $ pip install git+https://github.com/ARubiose/segwork.git

Registry

Classes for the implementation of a registry of components and a basic factory method.

class segwork.registry.ClassRegistry(register_hook: Optional[Callable] = None, get_hook: Optional[Callable] = None, initial_registry: Optional[MutableMapping] = None, *args, **kwargs)

Description

Maintains a registry of classes and provides a generic factory for instantiating them. Useful for modular components.

Args:
register_hook (typing.Callable): function that will receive the key and value as argument when registering a new item.
get_hook (typing.Callable): function that will receive the value as an argument when retrieving an item.
initial_registry (typing.MutableMapping): Dict-like object to initialize the registry with.

class segwork.registry.MutableRegistry(attr_name: Optional[str] = '_register_name', unique: bool = False)

Extends Registry with methods to add, modify and remove registered items.

Supports registering classes through decorators.

Args:

attr_name (str): If provided, register() will automatically detect the key to use when registering new classes. unique (bool): Determines what happens when two classes are registered with the same key:

  • True: The second class will replace the first one.

  • False: A ValueError will be raised

segwork.data.transform.generate_numpy_files(path: Union[str, Path], dataset: SegmentationDataset, color_map: MutableMapping[Tuple[int, int, int], int], index_name: bool = True)

Generate numpy files containing segmentation masks from PIL images

Args:
path (str, pathlib.Path): Output path for the numpy files.
dataset (SegmentationDataset): Dataset with labels as color images. It must implement the method load_label(idx)() to retrive labels.
color_map (dict): Dict-like object with color mappings.
index_name (bool) = True: If True generate files are named as 000.npy onwards.
